Einführung

Warum eine Proxmox Lizenz?

Ein Proxmox Abonnement gibt Ihnen Zugriff auf das Enterprise Repository mit stabilen, getesteten Updates und professionellem Support. Ohne Abonnement können Sie Proxmox VE vollständig kostenlos über das Community Repository nutzen — aber für Produktivumgebungen empfehlen wir immer ein Abonnement für die Stabilität und den Support.
Vergleich

Proxmox vs VMware vSphere Foundation

VMware verfolgt seit der Broadcom Übernahme ein per-Core Lizenzmodell von durchschnittlich €180 pro Core mit einer Mindestabnahme von 72 Cores — bei einem Server mit 64 Cores zahlen Sie schnell €11.000+ pro Jahr. Zusätzlich ist VMware nicht transparent mit ihren Preisen, sodass Sie nie genau wissen, was Sie zahlen müssen. Proxmox VE bietet alle Funktionen für einen festen Betrag pro CPU Socket.
  • VMware VVF: ~€180/Core, undurchsichtige Preise die sich regelmäßig ändern
  • Proxmox: €550/Socket fester Tarif, alle Features inbegriffen
  • VMware: vSAN als teures Add-on, nicht inbegriffen
  • Proxmox: HA, Ceph, Backup, SDN — alles inbegriffen, Open Source
Bestellen

Wie Bestellen?

Bestellen ist einfach. Wir sind offizieller Proxmox Partner und regeln die Lizenz direkt mit Proxmox. Sie erhalten den Subscription Key innerhalb von 24 Stunden.
  • Bestimmen Sie die Anzahl der CPU Sockets in Ihrer Umgebung
  • Wählen Sie das Support-Level (Community/Basic/Standard/Premium)
  • Kontaktieren Sie uns — wir erstellen ein Angebot
  • Wir aktivieren die Lizenz, Sie erhalten den Key
