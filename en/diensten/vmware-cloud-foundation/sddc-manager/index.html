What is SDDC Manager?

SDDC Manager is the central management system for VMware Cloud Foundation. It automates the implementation, configuration, management, and lifecycle of the complete software-defined data center (SDDC) stack - vSphere, vSAN, NSX, and Aria Suite.

SDDC Manager Architecture

Management Plane The brain of VCF:

  • API Server: REST API for external integrations
  • Workflow Engine: Automated deployment workflows
  • Inventory Service: Real-time inventory of all components
  • Health Service: Continuous health monitoring

Control Plane Central orchestration:

  • Configuration Management: Central configuration storage
  • Update Manager: Lifecycle management coordinator
  • Compliance Engine: Validation of configuration state
  • Backup Service: Configuration backup and restore

Data Plane Integration Connectivity with SDDC components:

  • vSphere: vCenter Server integration
  • vSAN: Storage policy management
  • NSX: Network and security configuration
  • Aria Suite: Monitoring and automation

Workload Domains

Concept

Logical separation of resources:

  • Management Domain: SDDC Manager and management VMs
  • Workload Domains: Production workload environments
  • VI Workload Domain: Traditional vSphere workloads
  • VNF Workload Domain: Network function virtualization

Benefits

  • Resource Isolation: Separate clusters per tenant/application
  • Independent Lifecycle: Updates without impact on other domains
  • Multi-Tenancy: Shared infrastructure with separation
  • Granular Control: Specific policies per domain

Domain Components

Each workload domain contains:

  • vSphere Cluster: Compute resources
  • vSAN Datastore: Storage resources
  • NSX Edge Cluster: Network services
  • Aria Suite: Monitoring per domain
