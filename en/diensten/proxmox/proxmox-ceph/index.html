Proxmox Ceph

Enterprise-grade software defined storage without extra license costs. The perfect replacement for VMware vSAN.

Belangrijkste Features

Ontdek de krachtige mogelijkheden voor uw infrastructuur.

CephFS

Distributed file system for shared storage between multiple nodes.

Self-Healing Storage

Automatic recovery of data in case of disk or node failures.

RADOS Block Device

Storage backend for virtual machines and containers with high performance and redundancy.

Included in Proxmox VE

Ceph storage is standard integrated in Proxmox VE without extra license costs.

Horizontally Scalable

Easily scale from a few nodes to hundreds of nodes with linear performance.

No Single Point of Failure

Fully distributed storage architecture with automatic redundancy.

Wat is Ceph?

Ceph is een opensource, software-defined storage platform dat object, block en file storage biedt. Geïntegreerd in Proxmox VE biedt het een krachtige storage oplossing zonder extra licentiekosten.

Ceph in Proxmox VE

Ceph is volledig geïntegreerd in Proxmox VE en biedt:

  • RADOS Block Device (RBD) - Voor VM en container storage
  • CephFS - Distributed file system
  • Object Storage (RGW) - S3 compatible object storage

Voordelen van Ceph in Proxmox

Geen Extra Licentiekosten

In tegenstelling tot VMware vSAN zijn er geen extra kosten per CPU of TB storage. Ceph is gratis inbegrepen bij Proxmox VE.

Schaalbaarheid

  • Van 3 nodes tot honderden nodes
  • Linear scale-out performance
  • Geen single point of failure

Flexibiliteit

  • Gebruik commodity hardware
  • Mix van SSD en HDD mogelijk
  • Thin provisioning
