VMware vSAN

Enterprise storage virtualisatie geïntegreerd in vSphere. Software-defined storage met industry-leading features.

Belangrijkste Features

vSAN biedt enterprise-grade storage mogelijkheden direct in uw hypervisor

Storage Policies

Policy-based storage per VM of workload voor flexibele prestaties en beschikbaarheid.

High Availability

Bescherm workloads met redundante storage en automatische herverdeling bij storingen.

All-Flash Performance

Hoge storage prestaties voor bedrijfskritische workloads en moderne applicaties.

Deduplicatie & Compressie

Verhoog storage-efficiëntie met deduplicatie en compressie op ondersteunde all-flash configuraties.

Encryptie at Rest

Bescherm data op storage devices met vSAN encryptie en KMS-integratie.

Stretched Clusters

Verhoog beschikbaarheid over twee locaties met metro-cluster architectuur.

Wat is VMware vSAN?

VMware vSAN is de software-defined storage laag binnen VMware Cloud Foundation (VCF). Het combineert lokale disks van ESXi hosts tot één gedeelde, gedistribueerde datastore die beschikbaar is voor het volledige cluster.

Met vSAN kunnen organisaties traditionele SAN of NAS storage vervangen door hyperconverged storage die direct geïntegreerd is in de hypervisor.

De rol van vSAN binnen VMware Cloud Foundation

Binnen VMware Cloud Foundation vormt vSAN de storage laag van het platform.

Samen met:

  • vSphere (compute virtualisatie)
  • NSX (network en security virtualisatie)

vormt vSAN een volledig software-defined datacenter.

Belangrijkste kenmerken:

  • Storage direct geïntegreerd in de hypervisor
  • Geen externe SAN infrastructuur nodig
  • Centraal beheer via vCenter
  • Automatische data redundantie en self-healing

Belangrijkste functionaliteiten

Policy-Based Storage

Beheer storage per workload via storage policies.

  • Bescherming en performance per VM
  • RAID-1 mirroring of RAID-5/6 erasure coding
  • Thin provisioning en QoS instellingen

Hyperconverged Storage

Compute en storage draaien op dezelfde hosts.

  • Geen aparte storage arrays nodig
  • Lagere infrastructuurkosten
  • Eenvoudigere schaalbaarheid

All-Flash Performance

Ondersteuning voor all-flash en NVMe configuraties voor maximale performance.

  • Lage latency
  • Hoge IOPS voor database en applicatie workloads
  • Optimale prestaties voor moderne datacenter workloads

Deduplicatie & Compressie

Verhoog storage efficiëntie door redundante data te reduceren.

  • Inline deduplicatie
  • Compressie van storage blocks
  • Lagere storage kosten

Data Encryptie

Bescherm data met vSAN encryptie op storage niveau.

  • Encryptie van data-at-rest
  • Integratie met externe Key Management Servers (KMS)

Stretched Clusters

Ondersteuning voor storage clusters over twee locaties.

  • RPO = 0 disaster recovery
  • Automatische failover
  • Metro cluster architectuur

Voordelen van vSAN

Lagere infrastructuurkosten

  • Geen externe SAN storage nodig
  • Minder hardware complexiteit
  • Lagere operationele kosten

Eenvoudiger beheer

  • Storage geïntegreerd in vSphere
  • Centraal beheer via vCenter
  • Policy-based configuratie

Schaalbaarheid

  • Voeg eenvoudig nieuwe nodes toe
  • Lineaire schaalbaarheid van capaciteit en performance
